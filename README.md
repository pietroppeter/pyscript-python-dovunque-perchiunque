## slides

slides fatte in [nim] con [nimib] e [nimislides]

### shortcut per me

per vedere le slide in locale (fatti un favore e usa [uv])

    uv run python -m http.server

poi:

    http://localhost:8000/index.html

o cose tipo:

    http://localhost:8000/chistacc%C3%A0.html
